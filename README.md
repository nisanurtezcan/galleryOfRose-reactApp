# React + Vite Project: Rose's Gallery
**LIVE LINK:** https://nisanurtezcan.github.io/galleryOfRose-reactApp/ 

This was my first React project. Even though it’s a small one, it helped me a lot in understanding React’s component logic. Creating a website for my sister’s art also motivated me to continue my React journey. Next, I’ll be practicing useState and other React hooks.

---

## 🛠️ Technologies Used

- CSS  
- JavaScript
